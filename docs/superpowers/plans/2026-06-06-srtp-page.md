# SRTP Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement a persistent `/srtp` page where users record itemized SRTP credit sources and see pass/excellent progress.

**Architecture:** Add a backend `srtp_records` table and authenticated CRUD API that returns record lists with backend-calculated summary. Replace the `/srtp` placeholder with a Vue page containing a summary dashboard, record cards, and create/edit modal.

**Tech Stack:** Express 5, Drizzle ORM, PostgreSQL, Zod, Vitest, Supertest, Vue 3, Vue Router, TanStack Vue Query, Tailwind CSS, Vue Test Utils.
